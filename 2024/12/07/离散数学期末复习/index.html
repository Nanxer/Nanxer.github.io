4.归纳定义法

  • 用这种方法定义一个非空集合A时，一般包含以下三步：

  • i）基本项

  • 已知某些元素（常用S0S0S0表示由这些元素组成的非空集合）属于A（即S0⊆AS0 \subseteq AS0⊆A）。这是构造A的基础，并保证A*不空。

  • ii）归纳项

  • 给出一组归纳规则，从A中的元素出发，依据这些规则所获得的元素，仍然是A中的元素。这是构造A的关键步骤。

  • iii）极小项

  • 如果集合S⊆AS \subseteq AS⊆A也满足i)和ii)，则S=AS=AS=A。这说明，A中每个元素都可以通过有限次使用i)和ii)来获得，他保证所构造出的集合A是唯一的。

5.皮亚诺公理

P1：0 ∈ N；

P2：若 n ∈ N, 则 n 有唯一的后继 n+ ∈ N；

P3：若 n ∈ N, 则 n+ = 0；

P4：若 n,m ∈ N 且 n+ = m+, 则 n = m；

P5：若 S ⊆ N 满足

i) 0 ∈ S；

ii) 如果 n ∈ S, 时 n+ ∈ S；

则 S = N。

第二章 二元关系

1.定义域与值域

设集合 A、B 为任意集合，R ⊆ A × B，令

domR=x∣x∈A且有y∈B使⟨x,y⟩∈RdomR = {x|x ∈ A且有y ∈ B使⟨x, y⟩ ∈ R}domR=x∣x∈A且有y∈B使⟨x,y⟩∈R

ranR=y∣y∈B且有x∈A使⟨x,y⟩∈RranR = {y|y ∈ B且有x ∈ A使⟨x, y⟩ ∈ R}ranR=y∣y∈B且有x∈A使⟨x,y⟩∈R

分别称 domR 和 ranR 为关系 R 的定义域和值域。

对任意的二元关系 R ⊆ A × B：

  1. domR 是 R 中所有序偶的第 1 维分量所构成的集合。

  2. ranR 是 R 中所有序偶的第 2 维分量所构成的集合。

2.关系的性质

设关系 R 为集合 A 上的二元关系。

  1. 若对每个a∈Aa\in Aa∈A，皆有 ⟨a,a⟩∈R⟨a, a⟩ ∈ R⟨a,a⟩∈R，则称 R 为自反的。

  2. 若对每个a∈Aa\in Aa∈A，皆有 ⟨a,a⟩∉R⟨a, a⟩ \notin R⟨a,a⟩∈/R，则称 R 为反自反的。

  3. 对任意的 a,b∈Aa,b\in Aa,b∈A，若当⟨a,b⟩∈R⟨a, b⟩ ∈ R⟨a,b⟩∈R 时皆有⟨b,a⟩∈R⟨b, a⟩ ∈ R⟨b,a⟩∈R，则称R 为对称的。

  4. 对任意的 a,b∈Aa,b\in Aa,b∈A若当⟨a,b⟩∈R⟨a, b⟩ ∈ R⟨a,b⟩∈R 且⟨b,a⟩∈R⟨b, a⟩ ∈ R⟨b,a⟩∈R 时皆有a = b，则称 R 为反对称的。

  5. 对任意的 a,b,c∈Aa,b,c\in Aa,b,c∈A，若当⟨a,b⟩∈R⟨a, b⟩ ∈ R⟨a,b⟩∈R 且⟨b,c⟩∈R⟨b, c⟩ ∈ R⟨b,c⟩∈R时皆有⟨a,c⟩∈R⟨a, c⟩ ∈ R⟨a,c⟩∈R，则称 R 为传递的。
