4.归纳定义法

  • 用这种方法定义一个非空集合A时，一般包含以下三步：

  • i）基本项

  • 已知某些元素（常用\(S0\)表示由这些元素组成的非空集合）属于A（即\(S0 \subseteq A\)）。这是构造A的基础，并保证A*不空。

  • ii）归纳项

  • 给出一组归纳规则，从A中的元素出发，依据这些规则所获得的元素，仍然是A中的元素。这是构造A的关键步骤。

  • iii）极小项

  • 如果集合\(S \subseteq A\)也满足i)和ii)，则\(S=A\)。这说明，A中每个元素都可以通过有限次使用i)和ii)来获得，他保证所构造出的集合A是唯一的。

2.集合的运算

设 A、B 为任意两集合，集合运算如下：

并运算：\(A∪B=\{x|x∈A或x∈B\}\)；

交运算：\(A ∩ B = \{x|x ∈A且x ∈B\}\)；当 A ∩ B = ∅ 时，称 A 与 B 不相交；

差运算：\(A−B=\{x|x∈A且x\notin B\}\)；

对称差运算：\(A⊕B=\\{x|(x∈A且x\notin B)或(x∈B且x\notin A)\\}=A∪B−A∩B\)；

补集：\(∼A= \\{x|x∈U且x\notin A\\}= U−A\)

德·摩尔根律

\[ \sim(A\cup B)=\sim A\cap \sim B \\ \sim(A\cap B)=\sim A\cup \sim B \]

4.自然数的定义（冯·诺伊曼方案）

  1. 0 ∈ N（注意，0 = ∅）；

  2. 若 n ∈ N，则 n+ ∈ N；

  3. 若 S ⊆ N 满足：

（1）0 ∈ S 且；

（2）若 n ∈ S，则 n+ ∈ S，则 S = N。

\[ 0 = \varnothing \\ 1 = 0^+ =\{∅\} = \{0\}\\ 2 = 1^+ = \{∅, \{∅\}\} = \{0, 1\}\\ 3 = 2^+ = \{∅, \{∅\}, \{∅, \{∅\}\}\} = \{0, 1, 2\}\\ ······\\ n + 1 = n^+= \{∅,\{∅\}, \{∅, \{∅\}\}, · · · \} = \{0, 1, 2, . . . , n − 1, n\}\\ \]

5.皮亚诺公理

P1：0 ∈ N；

P2：若 n ∈ N, 则 n 有唯一的后继 n+ ∈ N；

P3：若 n ∈ N, 则 n+ = 0；

P4：若 n,m ∈ N 且 n+ = m+, 则 n = m；

P5：若 S ⊆ N 满足

            i) 0 ∈ S；

            ii) 如果 n ∈ S, 时 n+ ∈ S；

        则 S = N。

6.归纳法

（第一归纳法）设 \(n_0 ∈ N\)，若对每个\(n∈\overline{N}_{n_0}\)，皆有一个命题P(n) 满足：

1._P_(\(n_0\)) 真；

2.对任意的 \(n∈\overline{N}_{n_0}\)，若 P(n) 真，则 P(n+) 也真。

则对任意的\(n∈\overline{N}_{n_0}\)，P(n) 皆真。

（第二归纳法）设\(n_0 ∈ N\)，若对每个\(n∈\overline{N}_{n_0}\)，皆有一个命题P(n) 满足：

1._P_(\(n_0\)) 真；

2.对任意自然数 \(n∈\overline{N}_{n_0}\)，若当\(k\in N且n_0\le k\lt n\)时，\(P(k)\)皆真，则 P(n) 也真。

则对任意的\(n∈\overline{N}_{n_0}\)，P(n) 皆真。

7.笛卡尔乘积

设\(n\in I_+\)且\(A_1,A_2,···,A_n\)为n个任意集合。若令

\[ A_1×A_2× · · · ×A_n= \{⟨x_1,x_2,· · ·,x_n⟩|x_i∈A_i,1⩽i⩽n\} \]

则称 \(A_1×A_2× · · · ×A_n\) 为 \(A_1×A_2× · · · ×A_n\) 的笛卡尔乘积，简记为\(\prod_{i=1}^n A_i\)，称 n为\(A_1×A_2× · · · ×A_n\)的维数。特别地，当\(A_1=A_2=···=A_n\)时，\(A_1×A_2× · · · ×A_n\)简记为\(A^n\)

第二章 二元关系

1.定义域与值域

设集合 A、B 为任意集合，R ⊆ A × B，令

\(domR = {x|x ∈ A且有y ∈ B使⟨x, y⟩ ∈ R}\)

\(ranR = {y|y ∈ B且有x ∈ A使⟨x, y⟩ ∈ R}\)

分别称 domR 和 ranR 为关系 R 的定义域和值域。

对任意的二元关系 R ⊆ A × B：

  1. domR 是 R 中所有序偶的第 1 维分量所构成的集合。

  2. ranR 是 R 中所有序偶的第 2 维分量所构成的集合。

2.关系的性质

设关系 R 为集合 A 上的二元关系。

  1. 若对每个\(a\in A\)，皆有 \(⟨a, a⟩ ∈ R\)，则称 R 为自反的。

  2. 若对每个\(a\in A\)，皆有 \(⟨a, a⟩ \notin R\)，则称 R 为反自反的。

  3. 对任意的 \(a,b\in A\)，若当\(⟨a, b⟩ ∈ R\) 时皆有\(⟨b, a⟩ ∈ R\)，则称R 为对称的。

  4. 对任意的 \(a,b\in A\)若当\(⟨a, b⟩ ∈ R\) 且\(⟨b, a⟩ ∈ R\) 时皆有a = b，则称 R 为反对称的。

  5. 对任意的 \(a,b,c\in A\)，若当\(⟨a, b⟩ ∈ R\) 且\(⟨b, c⟩ ∈ R\)时皆有\(⟨a, c⟩ ∈ R\)，则称 R 为传递的。

3.压缩与延拓

设 R 为集合 A 上的二元关系且 S ⊆ A，则称 S 上的二元关系R ∩ (S × S) 为 R 在 S 上的压缩，记为 R|S，并称 R 为 R|S 在 A上的延拓。

5.一些等价

设\(n ∈ I_+\)，R 为有限集合 \(A = \{x_1, x_2, · · · , x_n\}\)上的二元关系，\(a_{ij}（1 ⩽ i, j ⩽ n）\)为 R 的关系矩阵 MR 中第 i 行第 j 列的元素。

以下条件等价：

  1. R 是自反的；

  2. MR 的对角线元素全为 1；

  3. GR 中每个节点都有自圈。

以下条件等价：

  1. R 是反自反的；

  2. MR 的对角线元素全为 0；

  3. GR 中每个节点都无自圈。

以下条件等价：

  1. R 是对称的；

  2. MR 是对称矩阵；

  3. GR 中任意两个节点间若有边关联，则必为成对出现的反向有向边。

以下条件等价：

  1. R 是反对称的；

  2. 在 MR 中，若 1 ⩽ i，j ⩽ n 且 i = j，则 aij · aji = 0；

  3. GR 中任意两个节点间都无成对出现的反向有向边。

以下条件等价：

  1. R 是传递的；

  2. 在 MR 中，若有正整数 k ⩽ n 使 aik · akj = 1，则 aij = 1；

  3. 对 GR 中任意两个节点 xi 和 xj，若有从 xi 到 xj 的有向通路，则必有一条从 xi 到 xj 的有向边。即 GR 中处处有捷径。（从xi 到 xj 有有向通路，指的是从 xi 出发，沿着有向边箭头方向，经过若干个节点后能到达 xj。）
