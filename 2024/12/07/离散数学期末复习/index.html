4.归纳定义法

  • 用这种方法定义一个非空集合A时，一般包含以下三步：

  • i）基本项

  • 已知某些元素（常用\(S_0\)表示由这些元素组成的非空集合）属于A（即\(S_0 \subseteq A\)）。这是构造A的基础，并保证A*不空。

  • ii）归纳项

  • 给出一组归纳规则，从A中的元素出发，依据这些规则所获得的元素，仍然是A中的元素。这是构造A的关键步骤。

  • iii）极小项

  • 如果集合\(S \subseteq A\)也满足i)和ii)，则\(S=A\)。这说明，A中每个元素都可以通过有限次使用i)和ii)来获得，他保证所构造出的集合A是唯一的。

2.集合的运算

设 A、B 为任意两集合，集合运算如下：

并运算：\(A∪B=\{x|x∈A或x∈B\}\)；

交运算：\(A ∩ B = \{x|x ∈A且x ∈B\}\)；当 A ∩ B = ∅ 时，称 A 与 B 不相交；

差运算：\(A−B=\{x|x∈A且x\notin B\}\)；

对称差运算：\(A⊕B=\\{x|(x∈A且x\notin B)或(x∈B且x\notin A)\\}=A∪B−A∩B\)；

补集：\(∼A= \\{x|x∈U且x\notin A\\}= U−A\)

德·摩尔根律

\[ \sim(A\cup B)=\sim A\cap \sim B \\ \sim(A\cap B)=\sim A\cup \sim B \]

4.自然数的定义（冯·诺伊曼方案）

  1. 0 ∈ N（注意，0 = ∅）；

  2. 若 n ∈ N，则 n+ ∈ N；

  3. 若 S ⊆ N 满足：

（1）0 ∈ S 且；

（2）若 n ∈ S，则 n+ ∈ S，则 S = N。

\[ 0 = \varnothing \\ 1 = 0^+ =\{∅\} = \{0\}\\ 2 = 1^+ = \{∅, \{∅\}\} = \{0, 1\}\\ 3 = 2^+ = \{∅, \{∅\}, \{∅, \{∅\}\}\} = \{0, 1, 2\}\\ ······\\ n + 1 = n^+= \{∅,\{∅\}, \{∅, \{∅\}\}, · · · \} = \{0, 1, 2, . . . , n − 1, n\}\\ \]

5.皮亚诺公理

P1：0 ∈ N；

P2：若 n ∈ N, 则 n 有唯一的后继 n+ ∈ N；

P3：若 n ∈ N, 则 n+ = 0；

P4：若 n,m ∈ N 且 n+ = m+, 则 n = m；

P5：若 S ⊆ N 满足

            i) 0 ∈ S；

            ii) 如果 n ∈ S, 时 n+ ∈ S；

        则 S = N。

6.归纳法

（第一归纳法）设 \(n_0 ∈ N\)，若对每个\(n∈\overline{N}_{n_0}\)，皆有一个命题P(n) 满足：

1._P_(\(n_0\)) 真；

2.对任意的 \(n∈\overline{N}_{n_0}\)，若 P(n) 真，则 P(n+) 也真。

则对任意的\(n∈\overline{N}_{n_0}\)，P(n) 皆真。

（第二归纳法）设\(n_0 ∈ N\)，若对每个\(n∈\overline{N}_{n_0}\)，皆有一个命题P(n) 满足：

1._P_(\(n_0\)) 真；

2.对任意自然数 \(n∈\overline{N}_{n_0}\)，若当\(k\in N且n_0\le k\lt n\)时，\(P(k)\)皆真，则 P(n) 也真。

则对任意的\(n∈\overline{N}_{n_0}\)，P(n) 皆真。

7.笛卡尔乘积

设\(n\in I_+\)且\(A_1,A_2,···,A_n\)为n个任意集合。若令

\[ A_1×A_2× · · · ×A_n= \{⟨x_1,x_2,· · ·,x_n⟩|x_i∈A_i,1⩽i⩽n\} \]

则称 \(A_1×A_2× · · · ×A_n\) 为 \(A_1×A_2× · · · ×A_n\) 的笛卡尔乘积，简记为\(\prod_{i=1}^n A_i\)，称 n为\(A_1×A_2× · · · ×A_n\)的维数。特别地，当\(A_1=A_2=···=A_n\)时，\(A_1×A_2× · · · ×A_n\)简记为\(A^n\)

第二章 二元关系

1.定义域与值域

设集合 A、B 为任意集合，R ⊆ A × B，令

\(domR = {x|x ∈ A且有y ∈ B使⟨x, y⟩ ∈ R}\)

\(ranR = {y|y ∈ B且有x ∈ A使⟨x, y⟩ ∈ R}\)

分别称 domR 和 ranR 为关系 R 的定义域和值域。

对任意的二元关系 R ⊆ A × B：

  1. domR 是 R 中所有序偶的第 1 维分量所构成的集合。

  2. ranR 是 R 中所有序偶的第 2 维分量所构成的集合。

2.关系的性质

设关系 R 为集合 A 上的二元关系。

  1. 若对每个\(a\in A\)，皆有 \(⟨a, a⟩ ∈ R\)，则称 R 为自反的。

  2. 若对每个\(a\in A\)，皆有 \(⟨a, a⟩ \notin R\)，则称 R 为反自反的。

  3. 对任意的 \(a,b\in A\)，若当\(⟨a, b⟩ ∈ R\) 时皆有\(⟨b, a⟩ ∈ R\)，则称R 为对称的。

  4. 对任意的 \(a,b\in A\)若当\(⟨a, b⟩ ∈ R\) 且\(⟨b, a⟩ ∈ R\) 时皆有a = b，则称 R 为反对称的。

  5. 对任意的 \(a,b,c\in A\)，若当\(⟨a, b⟩ ∈ R\) 且\(⟨b, c⟩ ∈ R\)时皆有\(⟨a, c⟩ ∈ R\)，则称 R 为传递的。

3.压缩与延拓

设 R 为集合 A 上的二元关系且 S ⊆ A，则称 S 上的二元关系R ∩ (S × S) 为 R 在 S 上的压缩，记为 R|S，并称 R 为 R|S 在 A上的延拓。

5.一些等价

设\(n ∈ I_+\)，R 为有限集合 \(A = \{x_1, x_2, · · · , x_n\}\)上的二元关系，\(a_{ij}（1 ⩽ i, j ⩽ n）\)为 R 的关系矩阵 MR 中第 i 行第 j 列的元素。

以下条件等价：

  1. R 是自反的；

  2. MR 的对角线元素全为 1；

  3. GR 中每个节点都有自圈。

以下条件等价：

  1. R 是反自反的；

  2. MR 的对角线元素全为 0；

  3. GR 中每个节点都无自圈。

以下条件等价：

  1. R 是对称的；

  2. MR 是对称矩阵；

  3. GR 中任意两个节点间若有边关联，则必为成对出现的反向有向边。

以下条件等价：

  1. R 是反对称的；

  2. 在 MR 中，若 1 ⩽ i，j ⩽ n 且 i = j，则 aij · aji = 0；

  3. GR 中任意两个节点间都无成对出现的反向有向边。

以下条件等价：

  1. R 是传递的；

  2. 在 MR 中，若有正整数 k ⩽ n 使 aik · akj = 1，则 aij = 1；

  3. 对 GR 中任意两个节点 xi 和 xj，若有从 xi 到 xj 的有向通路，则必有一条从 xi 到 xj 的有向边。即 GR 中处处有捷径。（从xi 到 xj 有有向通路，指的是从 xi 出发，沿着有向边箭头方向，经过若干个节点后能到达 xj。）

6.关系的合成

设二元关系 R1 ⊆ A×B，R2 ⊆ B×C，则称从 A 到 C 的二元关系{⟨x, z⟩|有y ∈ B使⟨x, y⟩ ∈ R1且⟨y, z⟩ ∈ R2}为 R1 与 R2 的合成，并记为 R1 ◦ R2。

7.关系的闭包

设二元关系 R ⊆ A 2，如果二元关系 R ′ ⊆ A 2 满足：

  1. R ′ 是自反的（对称的，或传递的）；

  2. R ⊆ R ′；

  3. 若 A 上的二元关系 R ′′ 也满足 1) 和 2)，则 R ′ ⊆ R ′′。

就称 R ′ 为 R 的自反（对称，或传递）闭包，记为 r(R)（s(R)，或 t(R)）。

设二元关系 \(R ⊆ A^2\)，

  1. R 是自反的当且仅当 r(R) = R；

  2. R 是对称的当且仅当 s(R) = R；

  3. R 是传递的当且仅当 t(R) = R。

计算

设二元关系 R ⊆ A 2，

  1. \(r(R) = R ∪ I_A\)；

  2. \(s(R) = R ∪ R^{−1}\)；

  3. \(t(R) = R^+=\bigcup_{n=1}^{\infty}R^n\) 。

性质 设 A 为任意集合，二元关系 R ⊆ A 2

  1. rs(R) = sr(R)；

  2. rt(R) = tr(R)；

  3. st(R) ⊆ ts(R)。

8.相容关系

如果集合 A 上的二元关系 R 是自反的和对称的，则称 R 为 A 上的相容关系。若 xRy 则称 x 和 y 相容；否则称 x 和 y 不相容。

若 R 为集合 A 上的二元关系，则 R 为 A 上的相容关系，当且仅当 r(R) = s(R) = R。

相容类与最大相容类

设 R 为集合 A 上的相容关系。

  1. 若 S ⊆ A 且 S = ∅，如果对任意 x, y ∈ S 皆有 ⟨x, y⟩ ∈ R，则称 S 为相容关系 R 产生的一个相容类。

  2. 设 S 为 R 产生的相容类，若当 y ̸∈ S 时，皆有 x ∈ S 使得⟨x, y⟩ ̸∈ R，则称 S 为 R 的一个最大相容类。

9.覆盖与分块

设 A 为非空有限集合，若\(B ⊆ \Rho(A)\)且\(B = {A_i |1 ⩽ i ⩽ n且A_i ⊆ A}\) 满足

\(A_i \neq ∅（1 ⩽ i ⩽ n）\)；

\(\bigcup B = A\)。

则称 B 为集合 A 的覆盖，称 Ai ∈ B 为分块。

10.等价关系

如果集合 A 上的二元关系 R 是自反的、对称的和传递的，则称R 为 A 上的等价关系。

若 R 为集合 A 上的二元关系，则 R 为 A 上的等价关系，当且仅当 r(R) = s(R) = t(R) = R。

等价类

设 R 为集合 A 上的等价关系。对每个 x ∈ A，令[x]R = {y|y ∈ A且xRy}称 [x]R 为 x 关于 R 的等价类。当不强调 R 时，就把 [x]R 简记为[x]，并称为 x 的等价类。

11.拟序和半序

设 R 是集合 A 上的二元关系。

  1. 如果 R 是反自反的和传递的，则称 R 为 A 上的拟序关系，简称拟序，一般用“≺”表示，并称 ⟨A, R⟩ 为拟序结构。

  2. 如果 R 是自反的、反对称的和传递的，则称 R 为 A 上的半序关系，简称半序（偏序、部分序），一般用“≼”表示，并称 ⟨A, R⟩ 为半序结构。

关系

设 R 是集合 A 上的二元关系。

  1. 若 R 为 A 上的拟序，则 r(R) 是 A 上的半序。

  2. 若 R 为 A 上的半序，则 R − IA 是 A 上的拟序。

覆盖

设 R 是集合 A 上的半序且 a ∈ A，若 b ∈ A 满足：

  1. b \(\neq\) a 且 aRb；

  2. 若 x ∈ A 使 aRx 且 xRb，则必有 x = a 或 x = b。

就称 b 为 a 关于 R 的覆盖。在不特别强调关系时，也往往简称“b 为 a 的覆盖”。

元素 a 的覆盖 b，指的是在半序关系下，a 的“直接上级”b，即在半序关系下，元素 a 和 b 之间没法再插入其他元素。

12.全序

如果集合 A 上的半序 R 满足：若 x, y ∈ A，则 xRy 或 yRx。就称R 为 A 上的全序关系，简称全序（或线性序），并称 ⟨A, R⟩ 为链或全序结构（或线性有序集）。

13.极（最）大（小）元

设 ≼ 为非空集合 A 上的半序，S ⊆ A 且 S = ∅。

  1. 若有 a ∈ S，使得当 x ∈ S 且 a ≼ x（或 x ≼ a）时皆有x = a，就称 S 有极大 (或极小) 元，并称 a 为 S 的一个极大（或极小）元。

  2. 若有 a ∈ S，使得对每个 x ∈ S 皆有 x ≼ a（或 a ≼ x），就称S 有最大 (或最小) 元，并称 a 为 S 的一个最大（或最小）元。

14.上（下）（确）界

设 ≼ 为非空集合 A 上的半序，S ⊆ A 且 S = ∅。

若有 a ∈ A，使得对每个 x ∈ S 皆有 x ≼ a（或 a ≼ x），就称S 有上 (或下) 界，并称 a 为 S 的一个上 (或下) 界。

若有 S 的上 (或下) 界 a，使得对 S 的每个上 (或下) 界 y 皆有 a ≼ y（或 y ≼ a），就称 S 有上 (或下) 确界，并称 a 为 S的一个上 (或下) 确界，记为 a = supS（或 a = infS）。

15.良序

设 R 为非空集合 A 上的半序，如果 A 的每个非空子集都有最小元，就称 R 为 A 上的良序关系，简称良序，并称 ⟨A, R⟩ 为良序结构。也称集合 A 为良序集合。

第三章 函数

1.像与原像

设 f 为从集合 X 到 Y 的部分函数，A ⊆ X 且 B ⊆ Y。令

f[A] = {y|有x ∈ A使y = f(x)}

f −1 [B] = {x|有y ∈ B使f(x) = y}

称 f[A] 为 A 在 f 下的像，f −1 [B] 为 B 在 f 下的原像。

2.性质

设 f 为从集合 X 到集合 Y 的部分函数。

  1. 若 A1 ⊆ A2 ⊆ X，则 f[A1] ⊆ f[A2]；

  2. 若 B1 ⊆ B2 ⊆ Y，则 \(f^{−1} [B1] ⊆ f^{−1} [B2]\)；

  3. 若 A ⊆ domf，则 \(A ⊆ f^{−1} [f[A]]\)；

  4. 若 B ⊆ ranf，则 \(B = f[f^{−1} [B]]\)。

设 f 为从集合 X 到集合 Y 的部分函数，A ⊆ P(X)，B ⊆ P(Y)。

  1. f[∪A] = ∪ {f[A]|A ∈ A}；

  2. 若 \(A \neq ∅\)，则 f[ ∩ A] ⊆ ∩{f[A]|A ∈ A}；

  3. \(f^{−1} [ ∪ B] = ∪ \{ f^{−1} [B]|B ∈ B\}\)；

  4. 若 B ̸= ∅，则 \(f^{−1} [ ∩ B] = ∩ \{f^{−1} [B]|B ∈ B\}\)。

3.特殊函数

设 f 为从集合 X 到 Y 的部分函数。

  1. 若 domf = X，则称 f 为从 X 到 Y 的全函数，简称 f 为从 X到 Y 的（全）函数，记为 X −→ f Y 或 f : X → Y。

  2. 若 domf ⊂ X，则称 f 为从 X 到 Y 的严格部分函数。

  3. 若 ranf = Y，则称 f 为从 X 到 Y 上的部分函数。

  4. 若 ranf ⊂ Y，则称 f 为从 X 到 Y 内的部分函数。

  5. 若对任意的 x1, x2 ∈ domf，当 x1 = x2 时皆有 f(x1) = f(x2)，则称 f 为从 X 到 Y 的 1 − 1 部分函数。

4.重要函数

设 f : X → Y

  1. 若 ranf = Y，则称 f 为满射（surjective function）。

  2. 若 f 是 1 − 1 的，则称 f 为内射（injective function）。

  3. 若 f 既是满射，又是内射，则称 f 为双射（bijetive function）。

5.合成函数

合成函数 g ◦ f 与合成关系 f ◦ g 表示同一个集合 历史形成的，具有其方便之处：

  1. 对合成函数 g ◦ f，当 z = (g ◦ f)(x) 时，必有 z = g(f(x))，g ◦ f 与 g(f(x)) 的这种顺序关系很自然。

  2. 对合成关系 f ◦ g，当 ⟨x, z⟩ ∈ f ◦ g 时，必有 y ∈ Y 使 ⟨x, y⟩ ∈ f且 ⟨y, z⟩ ∈ g。这种顺序也很符合关系合成序偶的选取顺序。

6.特殊函数与函数合成

设 f : X → Y 和 g : Y → Z。则

  1. 若 f 和 g 都是满射，则 g ◦ f 也是满射；

  2. 若 f 和 g 都是内射，则 g ◦ f 也是内射；

  3. 若 f 和 g 都是双射，则 g ◦ f 也是双射。

设 f : X → Y 和 g : Y → Z。则

  1. 若 g ◦ f 是满射，则 g 是满射；

  2. 若 g ◦ f 是内射，则 f 是内射；

  3. 若 g ◦ f 是双射，则 g 是满射且 f 是内射。简称“左满右内”。

7.逆函数

设 X 和 Y 为两个集合，f : X → Y。

  1. Y若有 g : Y → X 使 g ◦ f =\(I_X\)，则称 f 为左可逆的，并称 g 为f 的一个左逆函数，简称左逆。

  2. 若有 g : Y → X 使 f ◦ g =\(I_Y\)，则称 f 为右可逆的，并称 g 为f 的一个右逆函数，简称右逆。

  3. 若有 g : Y → X 使 g ◦ f =\(I_X\)，且 f ◦ g =\(I_Y\)，则称 f 为可逆的，并称 g 为 f 的一个逆函数，简称逆。

8.内射与左可逆

设 X 和 Y 为两个集合且 X = ∅，若 f : X → Y，则下列条件等价：

  1. f 为内射；

  2. f 左可逆；

  3. f 左可消去。即对任意集合 Z 及任意的 g : Z → X 和h : Z → X，当 f ◦ g = f ◦ h 时，皆有 g = h。

9.满射与右可逆

设 X 和 Y 为二集合且 X = ∅，若 f : X → Y，则下列条件等价：

  1. f 为满射；

  2. f 右可逆；

  3. f 右可消去。即对任意集合 Z 及任意的 g : Y → Z 和h : Y → Z，当 g ◦ f = h ◦ f 时，皆有 g = h。

10.双射与可逆

设 X 和 Y 为二集合，若 f : X → Y 既是左可逆的，又是右可逆的，则 f 是可逆的，且 f 的左逆和右逆都等于 f 的唯一逆。

性质

若 X 和 Y 为二集合且 f : X → Y，则下列条件等价：

  1. f 是双射；

  2. f 既是左可逆的，又是右可逆的；

  3. f 是可逆的；

  4. f 的逆关系 \(f^{-1}\) 即为 f 的逆函数

11.特征函数

设 U 为全集，A ⊆ U，χA 为如下定义的从 U 到 R 的函数：

\[ \chi_A(x)=\begin{cases}1&x\in A\\0&x\notin A \end{cases} \]

称 χA 为集合 A 的特征函数。

  1. 0 是序数；

  2. 若 a 是序数，则 a + 也是序数；

  3. 若 S 是序数的集合（即 S 的元素都是序数），则 ∪ S 是一个序数；

  4. 任意一个序数都是经 1)-3) 获得的。

N 是一个序数

13.可良序

设 S 为任一集合，若 R 是 S 上的一个良序，则称集合 S 是由关系 R 所良序的。对于任意一集合 S，如果有一关系 R，使得 S 是由 R 所良序的，就称集合 S 是可良序的，或称 S 是良序的。

14.集合的基数

集合的势也称集合的基数，是用来衡量集合元素数量的量。

15.鸽笼原理/抽屉原理

通俗地表达为：如果把 n + 1 本书放进 n 个抽屉里，至少在一个抽屉里有两本或两本以上的书（鸽子）

16.无限集的性质

与 N 等势的集合称为可列集或可数集。

设 A 为集合。

  1. A 为无限集合；

  2. A 有可列子集；

  3. A 有与它等势的真子集。

第四章 命题逻辑

1.命题

具有确定真假意义的陈述句称为命题。由简单陈述句表示的判断称为简单命题，而由复合陈述句表示的判断称为复合命题。

2.符号化原则

基于符号集 Σ，命题逻辑的符号化原则一般为：

  1. 若命题为简单命题，可直接将其符号化为一个命题词；

  2. 若命题为复合命题，首先划分出构成该命题的所有简单命题，对每个简单命题，用不同的命题词进行符号化。其次，将复合命题中连词符号化为合适的逻辑联结词，用联结词连接命题词，即为复合命题的符号化结果；

  3. 在一般规则的基础上，与命题的含义相结合进行连词的符号化，选取合适的逻辑联结词；

  4. 可将符号化后的命题逻辑转换回自然语言表达的命题，对比二者的含义，可用来检查符号化结果是否正确。
