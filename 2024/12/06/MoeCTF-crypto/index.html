DA_BAI_TU

c=pow((a*p+b*q),t,N)， 二项式展开结合N=p*q，所以再mod N的情况下c=(a*p)^t+(b*q)^t mod N,因为其他项都含有p*q在mod N后都0。

结合c1与c2消去一个p(或q)，可以得到 q^(e1*e2) mod N的值，即q^(e1*e2)+k*N=q^(e1*e2)+k*p*q=q*(q^(e1*e2-1)+k*p)

并且q^(e1*e2-1)+k*p不含有p或者q的因子=>GCD(q^(e1*e2) mod N,N)=q,那么p和q都可以得到：

More_secure_RSA

n,N=>r，根据代码有m<2^128。

m^e mod N = C mod N 和 N = pqr 得到 m^e mod r = C mod r 同时显然有gcd(e, r-1)=1 （e为素数）

m mod r = C^dr mod r (dr=inverse(e, r-1))

r是1024bits，大于m，可有m = C^dr mod r。故可以在mod r下解决问题
