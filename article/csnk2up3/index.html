WSL搭配VScode配置sagemath环境

约 784 字大约 3 分钟

skillsagemathwslVisual Studio Code

2025-02-14

1.WSL的安装

参考网络教程(这里我选用的是微软商店里搜到的第一个ubuntu)

2.VScode的安装

参考网络教程，下载：Visual Studio Code - Code Editing. Redefined

3.Anaconda的安装

打开下载官网Download Now | Anaconda，下载linux版本的Anaconda~~（可以使用一些诸如果冻代理下载更快下载）~~

手动把安装文件移动到ubuntu中，并安装（基本只按回车就好）！最后的初始化建议选择yes！

或者在此之后再将其添加到环境变量中eval "$(/home/***/anaconda3/bin/conda shell.### hook)"（注：'***'为用户名称；'###'为shell名称，一般为bash，可以通过echo $0查看）

使用source ~/.bashrc更新一下环境，输入conda info检测一下是否安装成功

4.Sagemath的安装

运行如下指令（安装的Anaconda带有python3.12的环境，这也是选择安装Anaconda的原因之一）

conda config --add channels conda-forge
conda config --set channel_priority strict
conda create -n sage sage python=3.12

配置sage默认启动，不用每次都conda activate sage（可选）

打开新终端，默认就是sage环境

5.连接VScode

打开vscode安装wsl插件

远程连接ubuntu

先创建一个文件夹mkdir test，再打开

安装在WSL中的Sagemath插件

创建一个测试文件

SUCCESSFUL！

6.总结

参照sagemath官网Install from conda-forge - Installation Guide上用conda安装sagemath的方法。因为国内资源访问问题，改为手动安装Anaconda3。同时为了友好使用，采用VScode远程连接WSL的sage环境。
